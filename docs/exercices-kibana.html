Exercices Kibana

Accueil Hub exos

Exercices pratiques Kibana sur le dataset Shakespeare. Les solutions donnent les etapes UI et, quand utile, la requete Dev Tools de verification.

1) Creer le Data View Shakespeare

Objectif: creer un Data View sur shakespeare.

Voir la solution
Stack Management -> Data Views -> Create data view
Name: Shakespeare
Index pattern: shakespeare
Time field: none
Save

2) Explorer dans Discover

Afficher les colonnes play_name, speaker, speech_number, text_entry.

Voir la solution
Discover -> choisir Data View Shakespeare
Ajouter les champs en colonnes
Trier par speech_number desc
Sauvegarder la recherche: "Shakespeare QA"

3) Filtre KQL: lignes de HAMLET

Ecrire le filtre KQL correspondant.

Voir la solution
speaker: "HAMLET"

4) Filtre KQL: Macbeth + mot sleep

Trouver les lignes de Macbeth avec le terme sleep.

Voir la solution
play_name: "Macbeth" and text_entry: "sleep"

5) Lens: volume de lignes par piece

Graphique barres: X=play_name, metric=Count.

Voir la solution
Visualize Library -> Create visualization -> Lens
Drag play_name sur X
Metric: Count
Save: "Lines by play"

6) Lens: top speakers

Construire un top 10 des speakers.

Voir la solution
Create Lens chart
Axis: speaker
Metric: Count
Sort: Desc
Size: 10
Save: "Top speakers"

7) Lens Formula: part de Macbeth

Calculer la proportion de lignes appartenant a Macbeth.

Voir la solution
Formula:
count(kql='play_name: "Macbeth"') / count()

8) Dashboard multi-panels

Assembler 3 visuels dans un dashboard.

Voir la solution
Create dashboard
Add panels:
- Lines by play
- Top speakers
- Speech number distribution
Save dashboard: "Shakespeare Overview"

9) Ajouter des controls interactifs

Ajouter controls sur play_name et speaker.

Voir la solution
Dashboard -> Controls
Add options list: play_name
Add options list: speaker
Apply to all panels

10) Inspect request d un panel

Recuperer la requete exacte executee par Kibana.

Voir la solution
Sur un panel -> menu (...) -> Inspect -> Requests
Copier la requete
Coller dans Dev Tools pour verification

11) Rejouer en Dev Tools

Verifier le volume par piece en DSL.

Voir la solution
GET shakespeare/_search
{
  "size": 0,
  "aggs": {
    "by_play": {
      "terms": { "field": "play_name", "size": 20 }
    }
  }
}

12) Exporter les objets Kibana

Exporter Data View + visualisations + dashboard.

Voir la solution
Stack Management -> Saved Objects -> Export
Selectionner:
- Data view Shakespeare
- Visualisations Lens creees
- Dashboard Shakespeare Overview
Exporter en .ndjson

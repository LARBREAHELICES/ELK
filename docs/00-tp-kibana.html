Dataset shakespeare - parcours guidé pour construire un dashboard lisible.

Objectif

  1. Créer une visualisation Lens : répliques par pièce
  2. Créer une visualisation Lens : top speakers
  3. Créer une visualisation Lens : moyenne des années
  4. Assembler un dashboard final

Étape 1 - Vérifier les données

GET shakespeare/_count

Attendu : count > 0.

Étape 2 - Ouvrir Visualize Library

Analytics -> Visualize Library -> Create new visualization

Étape 3 - Choisir Lens

Dans la fenêtre de création, sélectionner Lens.

Étape 4 - Lens #1 : répliques par pièce

  • Type : Bar vertical
  • Horizontal axis : play_name
  • Vertical axis : Count of records

Étape 5 - Lens #2 : top 5 speakers

  • Horizontal axis : speaker
  • Vertical axis : Count
  • Tri : Descending
  • Limite : 5

Étape 6 - Lens #3 : moyenne des années

  • Horizontal axis : play_name
  • Vertical axis : Average(year)

Count = volume, Average = métrique calculée.

Étape 7 (option) - Timeline

7.1 Ajouter le mapping

PUT shakespeare/_mapping
{
  "properties": {
    "year_date": { "type": "date" }
  }
}

7.2 Alimenter year_date

POST shakespeare/_update_by_query
{
  "script": {
    "source": """
      if (ctx._source.year != null) {
        ctx._source.year_date = ctx._source.year + '-01-01T00:00:00Z';
      }
    """
  }
}

7.3 Config Lens

  • Type : Line
  • X-axis : Date histogram(year_date)
  • Y-axis : Count

Étape 8 - Dashboard final

Dans Dashboard -> Create, ajouter :

  • Répliques par pièce
  • Top speakers
  • Moyenne des années

Checklist de rendu

  1. Data View shakespeare prêt
  2. 3 visualisations Lens minimum
  3. 1 dashboard final
  4. Explication de chaque axe : horizontal = regroupement, vertical = calcul

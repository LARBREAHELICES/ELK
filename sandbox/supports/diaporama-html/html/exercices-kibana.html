Exercices Kibana

Accueil Hub exos

Exercices pratiques Kibana. Les solutions donnent les etapes UI et, quand utile, la requete Dev Tools de verification.

1) Creer le Data View Titanic

Objectif: creer un Data View sur titanic_data.

Voir la solution
Stack Management -> Data Views -> Create data view
Name: Titanic
Index pattern: titanic_data
Time field: none (si pas de date)
Save

2) Explorer dans Discover

Afficher les colonnes class, sex, survived, fare.

Voir la solution
Discover -> choisir Data View Titanic
Ajouter les champs en colonnes
Trier par fare desc
Sauvegarder la recherche: "Titanic QA"

3) Filtre KQL: femmes survivantes

Ecrire le filtre KQL correspondant.

Voir la solution
sex: "female" and survived: 1

4) Filtre KQL: Third class et fare bas

Trouver les passagers de 3e classe avec fare < 10.

Voir la solution
class: "Third" and fare < 10

5) Lens: taux de survie par classe

Graphique barres: X=class, metric=Average(survived).

Voir la solution
Visualize Library -> Create visualization -> Lens
Drag class sur X
Metric: Average of survived
Format metric: Percent
Save: "Survival rate by class"

6) Lens: breakdown par sexe

Ajouter sex en breakdown pour segmenter le resultat.

Voir la solution
Dans le visuel precedent:
Break down by: sex
Type: stacked bar
Comparer les segments female/male par classe

7) Lens Formula: part des femmes

Calculer proportion de femmes dans l echantillon.

Voir la solution
Formula:
count(kql='sex: "female"') / count()

8) Dashboard multi-panels

Assembler 3 visuels dans un dashboard.

Voir la solution
Create dashboard
Add panels:
- Survival rate by class
- Count by sex
- Fare histogram
Save dashboard: "Titanic Overview"

9) Ajouter des controls interactifs

Ajouter controls sur class et sex.

Voir la solution
Dashboard -> Controls
Add options list: class
Add options list: sex
Apply to all panels

10) Inspect request d un panel

Recuperer la requete exacte executee par Kibana.

Voir la solution
Sur un panel -> menu (...) -> Inspect -> Requests
Copier la requete
Coller dans Dev Tools pour verification

11) Rejouer en Dev Tools

Verifier le taux de survie par classe en DSL.

Voir la solution
GET titanic_data/_search
{
  "size": 0,
  "aggs": {
    "by_class": {
      "terms": { "field": "class" },
      "aggs": {
        "survival_rate": { "avg": { "field": "survived" } }
      }
    }
  }
}

12) Exporter les objets Kibana

Exporter Data View + visualisations + dashboard.

Voir la solution
Stack Management -> Saved Objects -> Export
Selectionner:
- Data view Titanic
- Visualisations Lens creees
- Dashboard Titanic Overview
Exporter en .ndjson
